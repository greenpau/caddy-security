# Security Policy

## Reporting a Vulnerability

To report a vulnerability, please send an email to greenpau@outlook.com.

When reporting, please include the following:

* Reproducable exploit of the reported vulnerability
* A draft of the security advisory for Github
* A draft of a blog post to be published at https://docs.authcrunch.com/. It must be
  interesting to read, include step by step walkthrough

> AI generated content is **NOT ALLOWED**, unless it was rewritten by a human who
> checked every line!
